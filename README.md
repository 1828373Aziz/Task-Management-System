# Task Management System (MVC Architecture)

A web-based Task Management System built with **PHP** following the **Model-View-Controller (MVC)** design pattern. This project was developed as part of the Software Engineering curriculum at the **University of Hail**.

## 🚀 Features
- **Dashboard:** Real-time statistics for total, completed, and pending tasks.
- **Task Management:** Full CRUD operations (Add, View, Complete, Delete).
- **Task Metadata:** Tasks include Priority levels (High, Medium, Low) and Due Dates.
- **Responsive UI:** Clean sidebar navigation and modern CSS styling.
- **MVC Design:** Separation of concerns between Data, Logic, and Presentation.

## 🛠️ Tech Stack
- **Backend:** PHP (PDO for secure database interaction)
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** MySQL
- **Architecture:** MVC (Model-View-Controller)

## 📂 Project Structure
- `/models`: Database interaction logic.
- `/controller`: Business logic and request handling.
- `/views`: UI templates and sidebar.
- `/config`: Database connection settings.
- `/CSS`: Application styling.

## ⚙️ Installation & Setup
1. Clone the repository to your local XAMPP `htdocs` folder.
2. Import the `task_db.sql` file into your MySQL database (phpMyAdmin).
3. Configure the database credentials in `config/database.php`.
4. Run the project via `localhost/task_mvc/index.php`.
